Purpose: Simulates a bank system in which multiple account and customer objects can be created, added into a bank class, and transactions be applied to with proper error checking. Prints out all information icluding when an account or customer is created, debit or credit has taken place, etc. with proper error messages returned and no memory leaks.

run "./a1" to run execuatable
